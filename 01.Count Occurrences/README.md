# Problem: Count Occurrences
## Description
Write a function that takes a string and a character and returns how many times that character appears in the string.

## Output
An integer representing the number of occurrences.

## Examples
```
Input: ("hello", "l")
Output: 2

Input: ("hello", "z")
Output: 0
```

## Constraints
Case-sensitive by default (you may implement case-insensitivity as an extension).
